# Claude Code Plugins

A curated collection of plugins for [Claude Code](https://claude.com/claude-code).

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [ux-review](https://github.com/GrillerGeek/ux-review) | Multi-agent UX review with simulated personas and specialist analysis | `claude install-plugin https://github.com/GrillerGeek/ux-review.git` |

## Installation

Each plugin is a standalone repository. Install any plugin directly:

```bash
claude install-plugin <plugin-url>
```

## Contributing

To add a plugin to this catalog, submit a PR adding a row to the table above.
